## Thiết lập database

1. chạy "XAMPP" và chọn "start MySQL"

2. mở "Navicat" và chọn "File" -> "Open External File" -> "Query" và chọn file "webs.sql" được giải nén từ "db.zip"

3. Chọn "Run" và database "web" đã được tạo nên

------------------------------------------------------------------------------------------------------------

## Triển khai web

1. dùng Terminal gọi "git clone https://github.com/huyhuyhuy8s/newshub.git"
hoặc truy cập: "https://github.com/huyhuyhuy8s/newshub" và chọn "Code" rồi "Download ZIP" và giải nén file.

2. Tạo một Terminal mới và gọi lệnh "npm i" để cài đặt các package cần thiết

3. Gọi lệnh "node main.js" để chạy trang web, trang web được chạy tại "http://localhost:3000/"
